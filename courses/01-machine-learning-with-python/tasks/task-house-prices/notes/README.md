# Notes Guide

Use these notes files as follows:

- `research-log.md` is the main canonical log for experiments, conclusions, and modeling decisions
- `learning-journal.md` can be used for informal reflections or study notes
- `submission-log.md` can be used for Kaggle or prediction-submission tracking

Rule going forward:
- do not create new ad hoc files like `decision.txt` or `experiments.txt`
- add new experiment conclusions to `research-log.md`
